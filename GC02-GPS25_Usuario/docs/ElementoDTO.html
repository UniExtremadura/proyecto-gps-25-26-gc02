Class: ElementoDTO

ElementoDTO()

new ElementoDTO()

DTO que representa un elemento del catálogo (canción, álbum, etc.).
Properties:
Name Type Description
id number ID único del elemento.
nombre string Nombre del elemento.
fechacrea string | null Fecha de creación en formato ISO (convertida desde LocalDateTime).
descripcion string | null Descripción del elemento.
urlFoto string | null URL de la imagen asociada al elemento.
numventas number | null Número total de ventas.
valoracion number | null Valoración media (0-5).
precio number | null Precio del elemento.
esnovedad boolean | null Indica si el elemento es una novedad.
esalbum boolean | null Indica si el elemento es un álbum (true) o una canción (false).
genero number | null ID del género principal.
subgenero number | null ID del subgénero.
artista number | null ID del artista creador.

Example
const elemento = new ElementoDTO({
  id: 12,
  nombre: "Midnight Dreams",
  fechacrea: "2024-02-10T15:30:00.000Z",
  descripcion: "Nuevo single exclusivo",
  urlFoto: "/uploads/elementos/12.jpg",
  numventas: 132,
  valoracion: 4.8,
  precio: 1.99,
  esnovedad: true,
  esalbum: false,
  genero: 3,
  subgenero: 5,
  artista: 9
});
