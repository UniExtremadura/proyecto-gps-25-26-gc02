(async, static) create(idusuario) → {Promise.<boolean>}

Registra una compra para un usuario: - Obtiene todos los elementos de la cesta del usuario. - Los inserta en `usuario_tiene_elemento`. - Vacía la cesta. Todo se ejecuta dentro de una transacción para evitar inconsistencias.
