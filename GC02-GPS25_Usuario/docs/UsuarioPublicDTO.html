Class: UsuarioPublicDTO

UsuarioPublicDTO()

new UsuarioPublicDTO()

DTO público del usuario, pensado para ser devuelto en respuestas públicas o en listados donde no deben incluirse datos sensibles.
Properties:
Name Type Description
id number | null ID del usuario.
nombreusuario string Nombre visible del usuario.
descripcion string | null Descripción pública del perfil.
rutafoto string | null Ruta de la foto de perfil.
esartista boolean Indica si el usuario es artista.

Example
const u = new UsuarioPublicDTO({
  id: 7,
  nombreusuario: "artistman",
  descripcion: "Artista de electro-pop",
  rutafoto: "perfil7.jpg",
  esartista: true,
});
